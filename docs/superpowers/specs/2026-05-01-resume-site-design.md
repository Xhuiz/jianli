# 单页个人简历网站设计规格

## 目标

在 `E:\JIANLI\index.html` 生成一个单文件个人简历网站。网站用于展示院耀辉的真实求职信息，目标岗位为 AI 训练师。页面必须简洁、高级、偏科技感，适配桌面端和移动端。

## 内容约束

页面只使用用户提供的真实信息，不添加头像、口号、评价语、虚构经历、虚构技能、虚构成果或外部图片。

必须展示的信息包括：

- 姓名：院耀辉
- 意向岗位：AI训练师
- 政治面貌：中共党员
- 年龄/籍贯：河北 22岁
- 意向城市：杭州｜随时到岗
- 电话：19531238160
- 邮箱：19531238160@163.com
- 身份：2025届毕业生｜未缴纳社保
- 核心优势、教育背景、实习经历、项目经历、校园经历、技能、证书、荣誉奖项全部按用户原文呈现

## 技术约束

- 单个 `index.html` 文件
- CSS 和 JavaScript 全部内联
- 不依赖 CDN、外部字体、外部图标库或图片
- 字体使用系统默认无衬线字体栈：`-apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif`
- 图标使用内联 SVG 或 CSS 绘制
- 动画使用 CSS transition/animation，不使用 JS 动画库

## 视觉风格

采用深色主题：

- 主背景：`#0a0a0a`
- 卡片背景：`#141414`
- 主强调色：`#3b82f6`
- 次要强调色：`#10b981`
- 主文字：`#e5e5e5`
- 次要文字：`#9ca3af`

整体参考 Vercel / Linear 的极简科技感：克制、清晰、信息密度适中。背景使用微妙蓝绿渐变线条和几何网格装饰，不使用大面积炫光或复杂装饰。

## 布局方案

采用 A. Executive Dashboard 布局。

桌面端：

- 顶部固定导航栏
- Hero 区域双栏：左侧姓名和身份信息，右侧联系方式卡片
- 核心优势使用 2x2 卡片
- 教育背景使用横向信息行和课程标签
- 实习/工作经历使用时间线
- 项目经历使用双卡片并排
- 校园经历使用紧凑时间线
- 技能证书使用双栏：左侧技能进度条，右侧证书列表
- 荣誉奖项使用网格卡片

移动端：

- 单栏布局
- 导航改为横向滚动
- 卡片和时间线保持可读，不出现文字重叠

## 交互行为

- 导航点击平滑滚动到对应区块
- 当前可视区块对应导航项高亮
- 滚动后导航栏从透明变成半透明深色毛玻璃
- 页面加载时各区块淡入上滑，依次延迟出现
- 技能进度条进入可视区域后从 0 动画到目标宽度
- 卡片 hover 时轻微上浮并增强阴影
- 页面右下角显示“回到顶部”按钮，滚动超过一屏后出现
- 电话使用 `tel:19531238160`
- 邮箱使用 `mailto:19531238160@163.com`

## 验收标准

- `E:\JIANLI\index.html` 可直接在浏览器打开
- 无外部网络依赖
- 桌面端双栏，移动端单栏
- 所有内容来自用户提供的真实信息
- 电话和邮箱可点击
- 导航、回到顶部、技能条动画正常
- 页面底部包含 `© 2025 院耀辉 · Built with AI`
